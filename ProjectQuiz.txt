What is a Content Delivery Network (CDN)?  List and explain at least FOUR advantages in using a CDN.

A content delivery network (CDN) is a system of servers that deliver webpages and content to a user based on user location, where the web page came from and a what server is being used.
Advantages:
1.Build-in version control - You can link an html or css file to it and request the lastest version of the file.
2.Different domains - CDNs are built on a different domain than the browser and can download multiple items at the same time.
3.No geographic barriers - It has a global reach and businesses can expand to other countries.
4.Saves money - Reduces the need to pay for foreign hosting because it is a single platform that can reach across regions.
